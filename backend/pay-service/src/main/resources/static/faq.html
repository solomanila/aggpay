常见问题  ·  FAQ
Q1 支持哪些支付方式？
支持 UPI、银行卡（Visa / Mastercard）、网银转账、电子钱包等，覆盖印度、印尼、泰国、巴西等多个市场。
Q2 资金到账需要多久？
UPI / 钱包：实时到账；银行卡：T+1 工作日；跨境汇款：1–3 工作日。具体以通道实际结算为准。
Q3 如何申请退款？
商家可通过管理后台发起退款，原路返回。处理时间 3–7 个工作日，取决于原支付通道。
Q4 手续费如何收取？
按交易流水收取服务费，费率根据合作协议而定。无隐藏费用，所有收费在账单中透明列示。
Q5 如何保障资金安全？
银行级加密，资金存放于监管账户，与运营资金完全隔离。实时风控系统拦截异常交易。
Q6 出现交易争议如何处理？
交易发生后 30 天内通过客服提交争议申请，附相关凭证，5 个工作日内响应处理。
Q7 如何接入 ZyaPay？
提供 RESTful API 和 SDK，支持主流开发语言。技术文档和沙箱环境可联系商务团队获取。
